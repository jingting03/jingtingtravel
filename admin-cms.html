🤖 AI 助理設定
如何取得 Claude API Key：
1. 前往 console.anthropic.com 註冊帳號
2. 點擊左側「API Keys」→「Create Key」
3. 複製 API Key（以 sk-ant- 開頭）貼到下方
4. 儲存設定後，前台 AI 助理就會變成真實的 Claude AI
⚠️ Key 儲存在你的瀏覽器，不會上傳到任何伺服器
📧 Mailchimp 訂閱整合
如何設定 Mailchimp：
1. 前往 mailchimp.com 免費註冊
2. 建立一個 Audience（聯絡人列表）
3. 到 Account → Extras → API Keys → Create A Key
4. 複製 API Key 和 Audience ID 填入下方
設定後，前台訂閱的 Email 會自動同步到 Mailchimp，你可以直接用 Mailchimp 發送週報
